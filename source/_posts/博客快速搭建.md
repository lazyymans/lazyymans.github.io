### Create a new post and commit

``` bash
cd 你的博客目录
$ hexo new "My New Post"	//  /你的博客目录/source/_posts/My New Post.md
git commit -am 'add My New Post.md and commit'
git push

hexo clean	//	先进行clean
hexo d -g	//	更新到master分支
```
